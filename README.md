# Github-Webhook-To-Kook-Card-Template
一份将 Github Commit 利用 Github Webhook 推送到 KOOK 的卡片消息模板

KOOK上线了[消息模板&管道](https://developer.kookapp.cn/doc/http/template)，于是乎我们可以利用这个来实现将 Github 项目的新 commit 的信息推送到 KOOK。
